# 1. 产品概述

## 1.1 产品定位

AI商业情报雷达是一款基于大语言模型（LLM）的智能商业信息分析平台。

产品通过自动采集互联网公开资讯，结合 AI
推理能力，对行业动态、商业机会和创业案例进行结构化分析，帮助用户快速理解行业变化并发现潜在机会。

核心价值：

> 将人工的信息搜集、筛选和分析流程，转化为自动化 AI 情报分析流程。

------------------------------------------------------------------------

# 2. 产品背景

## 2.1 用户痛点

当前信息获取存在以下问题：

### 信息过载

每天产生大量行业新闻、政策变化、商业案例，用户无法快速筛选有效信息。

### 信息价值判断困难

普通用户能够看到新闻，但难以判断：

-   哪些信息具有商业价值？
-   哪些趋势值得关注？
-   哪些机会可以实际执行？

### 人工分析成本高

传统方式需要：

    搜索资讯
    ↓
    阅读文章
    ↓
    整理信息
    ↓
    总结趋势
    ↓
    输出报告

整个过程耗费大量时间。

------------------------------------------------------------------------

# 5. 产品核心功能设计

------------------------------------------------------------------------

# 功能一：多源信息采集

## 功能描述

系统自动从多个信息源获取行业资讯。

## 数据来源

  来源         作用
  ------------ ------------------
  Tavily API   全球英文资讯搜索
  Bocha API    中文互联网搜索
  RSS          行业媒体订阅

------------------------------------------------------------------------

## 产品流程

    用户选择行业

    ↓

    系统生成搜索关键词

    ↓

    调用搜索接口

    ↓

    获取原始资讯

    ↓

    进入AI分析流程

------------------------------------------------------------------------

# 功能二：AI商业分析 Agent

## 功能描述

利用 DeepSeek V3 对原始资讯进行理解和推理。

## 输入

-   新闻标题
-   新闻内容
-   来源链接

## 输出

生成：

-   核心行业情报
-   趋势判断
-   商业机会
-   风险分析

------------------------------------------------------------------------

## Agent流程

    资讯数据

    ↓

    Prompt控制

    ↓

    LLM推理

    ↓

    结构化商业报告

------------------------------------------------------------------------

# 功能三：商业案例挖掘

## 功能描述

针对指定方向，自动寻找真实商业案例。

例如：

输入：

    AI知识库 自动化工作流

输出：

  分析维度   内容
  ---------- ------------
  案例定位   项目是什么
  商业模式   如何赚钱
  流量来源   如何获客
  复制路径   如何学习

------------------------------------------------------------------------

# 功能四：Streamlit应用展示

## 功能描述

提供 Web 端交互界面。

用户可以：

-   选择行业
-   输入访问权限
-   查看分析报告
