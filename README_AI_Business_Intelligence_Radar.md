## 项目介绍

AI商业情报雷达是一款面向创业者、AI从业者和内容创作者的智能商业信息分析平台。

项目通过自动采集全球及国内公开资讯，结合大语言模型进行信息理解、趋势分析和商业机会提炼，将分散的信息转化为结构化商业洞察报告。

核心目标：

> 利用 AI 自动完成「信息获取 → 内容理解 → 商业判断 → 报告生成」流程。

------------------------------------------------------------------------

# 产品背景

当前互联网信息环境存在：

-   信息来源高度分散
-   行业变化速度快
-   普通用户难以快速判断信息价值
-   人工整理行业动态成本较高

因此设计该系统，通过 AI Agent 工作流帮助用户降低信息筛选成本。

------------------------------------------------------------------------

# 产品功能

## 1. 多源资讯采集

系统支持多种信息来源：

  数据源   用途
  -------- --------------------
  Tavily   全球英文资讯搜索
  Bocha    中文互联网信息搜索
  RSS      行业媒体订阅

支持针对不同商业赛道进行定向检索。

------------------------------------------------------------------------

## 2. AI商业分析

通过 DeepSeek V3 大语言模型完成：

-   新闻摘要
-   行业趋势判断
-   商业价值分析
-   创业机会挖掘
-   风险因素识别

------------------------------------------------------------------------

## 3. 多行业分析场景

当前支持：

### 人工智能与大模型

分析：

-   AI创业机会
-   大模型商业化趋势
-   开源生态变化

### 自媒体与流量变现

分析：

-   内容平台变化
-   创作者经济机会
-   流量增长方向

### 跨境电商与出海

分析：

-   平台政策变化
-   出海机会
-   市场趋势

------------------------------------------------------------------------

## 4. AI商业案例挖掘

通过 `case_hunter.py` 实现：

输入目标方向：

    AI知识库 自动化工作流

自动：

    搜索案例
        ↓
    信息整理
        ↓
    DeepSeek商业分析
        ↓
    生成商业案例表格

输出：

-   商业模式
-   获客方式
-   收入逻辑
-   可复制路径

# 本地运行

## 1. 安装依赖

``` bash
pip install -r requirements.txt
```

## 2. 配置环境变量

复制：

    .env.example

填写：

    TAVILY_API_KEY=
    DEEPSEEK_API_KEY=
    BOCHA_API_KEY=

    FEISHU_WEBHOOK_URL=
    WECOM_WEBHOOK_URL=

    APP_ACCESS_CODE=
    ADMIN_ACCESS_CODE=

## 3. 启动应用

``` bash
streamlit run app.py
```
